# QuickMiner
First Runescape/powerbot mining script. *Must start with pickaxe equipped*

# Locations
* Varrock West
* Varrock East

# Rocks
* Tin
* Copper
* Clay
* Iron

# Banking
* Will bank when inventory is full
* Supports powermining

# Other Features
* Shift dropping (If enabled)
